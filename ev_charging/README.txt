1) Add folder ./datafev_package/src to your PYTHONPATH environment variable

2) Adjust Python script ./ev_profile_generation/generate_profiles_csv.py according to your needs and install all its package requirements (e.g. using pip)

3) If desired, adjust the EV statistics and assumptions in Excel file ./ev_profile_generation/Statistics.xlsx

4) Run Python script ./ev_profile_generation/generate_profiles_csv (Python >3.9 recommended)

5) The EV charging power time series are finally located in folder ./ev_profile_generation/profiles
